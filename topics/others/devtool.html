Entwicklerwerkzeuge

Hilfe vom Browser

Wenn man einfach mal nicht weiß wo das Problem ist helfen einem oft die Entwicklerwerkzeuge des Browsers weiter. Sie können ganz einfach mit dem Tastenkürzel "F12" aktiviert werden und der Aufbau ist in jedem Browser ähnlich. Aber auch ein Rechtsklick auf die Seite und "Element untersuchen" öffnet sie. Der erste Tab ist auch der, welcher für die HTML-CSS Entwicklung nützlich ist, die anderen sind erstmal uninteressant. Der Name ist von Browser zu Browser verschieden. Bei Firefox heißt er Inspektor und bei MS Edge wird er Elemente gennant. Hier kann man den kompletten Quellcode der Seite sehen und auch CSS-Code kann verändert werden. Es gibt auch ganz oben links eine kleine Schaltfläche, genannt "Element auswählen", wenn man diese betätigt und mit der Maus über die Seite fährt wird einem jedes einzelne Element in einer visuellen Box dargestellt. Sogar Außen- und Innenabstände sowie Ränder werden farbig hervorgehoben. Wenn man dann auf ein Element klickt werden einem der Code-Abschnitt sowie weitere Infos im Entwicklerwerkzeug angezeigt. Hier kann man auch live Änderungen vornehmen, allerdings sind diese sobald die Seite neu geladen wird wieder weg, d.h. wenn du eine passende Einstellung über das Entwicklerwerkzeug gemacht hast, musst du sie auch im Html-Code übernehmen. Es gibt sogar die Möglichkeit ein Smartphone zu simulieren um zu testen ob die Seite auch auf kleinen Geräten gut dargestellt wird.

Ein kleiner Tipp noch, aktiviere die Einstellung "eigenes Fenter", somit wird die Seite nicht in einem falschen Verhältnis dargestellt und das Fenster kann einfach weggeklickt werden sobald es nicht mehr gebraucht wird.

Hilfe vom Validator

Es gibt im Internet verschieden Validatoren, diese prüfen dein Html-Dokument auf Fehler und helfen dir dabei es vollständig gültig zu machen. Auch für CSS gibt es solche. Der Code-Editor "VSCode" hat übrigens eine Erweiterung welche, sobald du sie herunterlädst, dies für dich automatisch beim schreiben des Codes übernimmt.

  • Html-Validator
  • CSS-Validator

Lade hier einfach die fertige Datei hoch (oder gib die URL an, sofern du die Seite auf einem Server hosten lässt) und lasse sie kostenlos auf Fehler überprüfen.
