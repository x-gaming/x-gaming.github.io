Farben - colors

Farben haben eine besondere Wirkung und damit nicht alles so langweilig in Schwarz-Weiß ist, gibt es natürlich auch Farben in CSS. Es gibt dabei ganz unterschiedliche Wege eine Seite farbig zu gestalten. Egal ob nun der Text ansich, der Hintergrund oder geometrische Formen, irgendwie muss ein Wert für die Eigenschaft color angegeben werden.

Farben mit Worten

Wenn man eine der 8 Grundfarben (Rot, Grün, Blau, Cyan, Magenta, Gelb, Weiß oder Schwarz) nutzen möchte ist dies kein Problem. Man schreibt einfach den Namen auf englisch auf. Auch helle und dunkle Farbtöne sind kein Problem man kann ja einfach "dark" oder "light" davor setzen und schon hat man die Farbe definiert. Es gibt sogar Namen wie "olive" oder "navy" als Farbwert.

Farben mit Hex-Codes

Um aber auch wirklich jeden Farbton benutzen zu können gibt es die Farbcodes. Mit diesen kann man genauso Farben mischen wie mit dem nachfolgenden RGB-Model. Jedoch gibt es diese schon länger denn sie stammen aus der Anfangszeit von HTML.

Hex-Code Farbe
#000000 Schwarz:
#FFFFFF Weiß:
#FF0000 Rot:
#00FF00 Grün:
#0101DF Blau:
#FFFF00 Gelb:
#4C0B5F Violet:
#FF8000 Orange:

Farben mit dem RGB-System

In CSS gibt es das RGB-System, mit welchen es möglich ist 16,7 Millionen verschiedene Farbwerte einzustellen. Man nutzt hierfür die Funktion rgb(Rot, Grün, Blau);. Für die verschieden Werte setzt man dann eine Zahl zwischen 0 und 255 ein, je höher die gewählte Zahl desto mehr Anteil hat die jeweilige Farbe. Übrigens, wenn alle 3 Werte gleich groß sind kommt immer ein Grauton raus. Zur Veranschaulichung hier nochmal ein paar Beispiele.

RGB-Wert Farbe
rgb(0, 0, 0) Schwarz
rgb(255, 255, 255) Weiß
rgb(255, 0, 0) Rot
rgb(0, 255, 0) Grün
rgb(0, 0, 255) Blau
rgb(100, 100, 100) Grau

Farben mit dem RGBA-System

RGBA ist eine kleine Erweiterung des RGB-Systems, hier kommt noch der sogennate Alpha Kanal hinzu. Dieser ist für Transparenz verantwortlich und kann einen Wert von 0.0 bis 1.0 annehmen, wobei diese minmal und maximal Werte dem normalen RGB-System entsprechen und somit nicht genutzt werden. Wie man schon erkennt ist eine Kommazahl erforderlich und die CSS-Funktion ist wie folgt: rgba(Rot, Grün, Blau, Alpha); ein transparentes Schwarz hätte den Wert rgba(0, 0, 0, 0.5);.
