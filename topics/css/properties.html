Eigenschaften für CSS

Hier ist eine Übersicht mit Eigenschaften welche nach persönlicher Erfahrung immer / häufig gebraucht werden. Wenn du in der Spalte "Wie?" bei der jeweiligen Eigenschaft auf das "|-|" klickst bekommst du eine Auswahl an möglichen Werten. Da viele Eigenschaften ansich auch noch weitere spezielle Eigenschaften mitbringen wurden diese ebenfalls unter diesem Punkt aufgezählt.

Eigenschaften für Texte

Eigenschaft Funktion Wie?
color: legt die Schriftfarbe fest |-|
font-family: legt die Schriftart fest |-|
font-size: legt die Schriftgröße fest |-|
font-style: legt die Text-Darstellung fest |-|
letter-spacing: legt den Abstand zwischen den einzelnen Buchstaben fest |-|
word-spacing: legt den Abstand zwischen einzelnen Wörtern fest |-|

Eigenschaften für visuelle Darstellung

Eigenschaft Funktion Wie?
background: legt den Hintergrund fest |-|
border: legt den Rand fest |-|
box-shadow: erzeugt einen Schatten |-|
cursor: legt das Aussehen des Mauszeigers fest |-|
display: legt den Anzeigemodus fest |-|
position: legt die Art der Positionierung fest: |-|
overflow: legt das Verhalten fest wenn der Anzeigebereich zu klein für das Element ist |-|
z-index: legt die sichtbare Ebene fest |-|
siehe Layouts für sämtliche Layout Eigenschaften |-|

Eigenschaften für Abstände

Eigenschaft Funktion Wie?
margin: legt Abtand UM Element fest (Außenabstand) |-|
padding: legt Abstand um den INHALT des Elements fest (Innenabstand) |-|
outline: legt einen Rahmen fest welcher selbst keinen weiteren Platz benötigt |-|

Eigenschaften für dynamische Veränderungen

Eigenschaft Funktion Wie?
animation: erstellt eine neue Animation |-|
transition: ändert Werte einer Eigenschaft über eine bestimmte Zeit |-|
transform: ändert u.a. Größe und Position von Elementen |-|
perspective: legt die Perspektive fest |-|
