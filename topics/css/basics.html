CSS

(Cascading Style Sheets)

CSS ist eine Formatierungssprache und kann ohne HTML nicht funktionieren. Mit CSS legt man zum Beispiel Farben, Größen, oder das Layout fest. Um dies zu tun nutzt CSS sogenannte "Selektoren", diese haben jeweils verschiedene Eigenschaften welchen man wiederrum Werte zuweist. Ein Selektor bestimmt dabei immer welches Element (html-tag) angesprochen wird und die Eigenschaften legen dann das Aussehen fest.

Der Selektor

Als Selektor kann entweder der tag ansich, eine ID oder eine Klasse genutzt werden, auch eine Kombination aus allem ist möglich. Zum Beispiel kann man sagen, dass nur Klassen welche im Element body liegen verändert werden sollen. Oder nur ID´s welche in der Klasse "Beispiel" liegen.

ID & Klasse?

Der große Unterschied zwischen einer ID und einer Klase ist, dass eine ID immer nur für ein Element pro Seite genutzt wird. Das bedeutet eine ID macht ein Element einzigartig, hingegen zeichnet die Klasse mehrere Elemente aus. Kurz gesagt, ID für nur ein einziges Element, Klasse für mehrere Elemente.

Nun kennst du den Unterschied, aber wie definiere ich eine ID oder Klasse? Das ganze wird im HTML Code getan. Im Grunde sind beides auch nur Attribute eines tags und werden auch entsprechend definiert. D.h. Man schreibt hinter dem "Start-tag" direkt id = "idname" oder für eine Klasse eben class = "classname". Wichtig ist, dass man einen eindeutigen Namen, welcher am besten auch die Funktion noch beschreibt wählt.

Im CSS-Code wird eine ID dann über einen Hashtag (#) angesprochen und eine Klasse über einen ganz normalen Punkt. Übrigens ist es wichtig, dass man auf die Groß- und Kleinschreibung achtet, denn so wie man die ID oder Klasse definiert hat muss man sie auch "ansprechen".

Die Eigenschaft mit Wert

Eine Eigenschaft wird immer klein geschrieben und endet mit einem Doppelpunkt. Ein Leerzeichen grenzt sie dann von ihrem Wert ab, welcher wiederrum mit einem Semikolon endet. Wichtig ist, dass man pro Zeile im Dokument nur eine Eigenschaft festlegt.
