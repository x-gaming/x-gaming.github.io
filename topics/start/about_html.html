Über Html

(Hypertext Markup Language)

Hypertext Markup Language oder kurz HTML wird auf fast jeder Webseite als "Grundgerüst" verwendet. Die neuste Version 5.2 wird von jedem gängigen Browser unterstützt und diese sollte man auch verwenden. Um jedem Nutzer die beste Erfahrung zu bieten, sollte man aber natürlich auch auf Abwärtskompatibilität achten.

Html ist eine textbasierte Auszeichnungssprache und keine Programmiersprache wie manche vielleicht denken. Der entscheidende Unterschied ist hier, dass HTML nicht rechnen kann und keinerlei Funktionen ausführt, Html wird nur zum Strukturieren der Seite verwendet. Mit CSS (Cascading Style Sheets) oder anderen Style-Frameworks (z.B. Bootstrap) wird dann die Seite formatiert und ansprechend gestaltet, mit Javascript kann dann schließlich das Verhalten der Seite auf z.B. Nutzereingaben festgelegt werden.

Übrigens:

  1. Man kann mit einem Rechtsklick > Quelltext anzeigen (oder mit der Taste F12) den Html Code jeder Internetseite lesen und sogar lokal verändern (die Änderungen sind nach dem nächsten neuladen weg).
  2. Mit dem Tastenkürzel "Strg + S" kann man in vielen Browsern die gesamte Internetseite speichern und hat somit den Html Code + Stylesheets, jedoch sollte man keinen fremden Code einfach kopieren.

Wie man an oben genanten Fakten erkennen kann, ist Html nicht sehr sicher vor "Diebstahl" und deshalb sollte man persönliche Dinge (wie z.B. Passwörter) nie in den Html Code schreiben. Sicherheit
