Wie starte ich mit Html?

Schritt 1:

Wenn nicht schon vorhanden installierst du zuerst eine Softwareumgebung, natürlich kannst du auch den ganz normalen Windows/ Linux Texteditor verwenden aber es ist ratsam einen speziellen Code Editor zu nutzen. Dieser erleichtert dir nämlich die Arbeit mit z.B. besserer Text Formatierung, automatischen Vervollständigungen, farbig hervorgehobenen Textstellen oder anderen nützlichen Funktionen.

Hier sind drei Beispiele für Editoren (welche übrigens auch die meisten anderen gängigen Programmier- und Skriptsprachen unterstützen).

NotPad++

NotPad++: reicht für den Anfang völlig aus.

Download: https://notepad-plus-plus.org/

Codepen.io/

Codepen.io: Ist ein Online-Editor bei welchem man sich nur registrieren muss.

Seite: https://codepen.io/

Visual Studio Code

Visual Studio Code (VsCode): sieht etwas moderner aus und hat besonders für erweiterte Funktionen bessere Werkzeuge.

Download: https://code.visualstudio.com/

Schritt 2:

  1. Für die erste eigene Html Seite erstellt man zuerst einen neuen Ordner welchen man so bennent, dass man ihn später wiederfindet. Dieser Ordner wird der "Arbeitsplatz" und hier werden alle Dateien welche später auf der Seite gebraucht werden gespeichert.
  2. Nun im Ordner ein neues Textdokument erstellen.
  3. diese Datei mit dem zuvor gewählten Editor öffnen und unter einem passenden Namen (häufig: "index.html" oder "Startseite.html") abspeichern. Wichtig falls der Editor dies nicht übernimmt den Dateityp unbedingt auf "Alle Dateien" oder "*.*" ändern.
  4. Dies ist zwar optional, sollte aber gemacht werden um bei größeren Projekten den Überblick zu behalten:
    • Einen weiteren Ordner "html" für Unterseiten
    • Einen weiteren Ordner "pics" für Bilder
    • Einen weiteren Ordner "style" für CSS-Dateien
    • Einen weiteren Ordner "scripts" für JavaScript
    (Dies sind nur Beispielnamen und somit sind auch andere Namen okay. Außerdem sollten entsprechende Ordner nur erstellt werden, wenn gennante Dateien überhaupt geplant sind.)

Schritt 3:

Um zu testen ob vorherige Schritte funktioniert haben kannst du die erstellte .html Datei mit einem Browser deiner Wahl öffnen (idealerweise hat die Datei bereits das Logo von diesem angenommen und lässt sich per Doppelklick öffnen). Du solltest nun eine komplett weiße Seite sehen wo in der URL Leiste des Browsers der Pfad zu deiner Datei steht.
