Grafik, Video & Audio

Allgemeine Info

Grafiken, Videos und Audio machen Seiten direkt ansprechender und geben mehr Funktion, jedoch sollte man immer überlegen wo und vorallem wie viel man einfügen möchte. Zum einen haben sie den Vorteil, dass bestimmte Abschnitte genauer beschrieben werden können, anschauliche Beispiele gegeben werden und allgemein der erste Eindruck verbessert wird.

Aber es gibt auch Nachteile, welche vorallem bemerkbar werden wenn man auf die Performance der Seite achtet. Bilder, Videos oder Audio benötigt nämlich wesentlich mehr Speicherplatz als ein einfacher Text, dies hat zur Folge, dass die Seite allgemein langsamer läd und länger braucht bis sie vollständig dargestellt wurde. Wenn man zum Beispiel die Startseite direkt mit einer Bildergalerie ausstattet sollte einem bewusst sein, dass man abhängig von der Internetverbindung merkbar länger braucht um die Seite zu nutzen. Auch wenn man an Nutzer denken möchte welche Mobile Daten verwenden, sollte man dies berücksichtigen und vielleicht die Bildergalerie auf eine separate Unterseite verschieben.

Grafiken

Bilder bzw. Grafiken werden über den tag <img> eingefügt und mit dem Attribut "src" wird die Quelle zum Bild bereitgestellt. Dabei kann die Quelle auch ein Link zu einem Bild auf Google sein. Besonders hier ist es dann wichtig eine Alternative anzugeben. Mit dem Attribut alt wird dann z.B. ein Text angezeigt wenn das Bild nicht gefunden wurde oder anderweitige Probleme vorliegen.

Es gibt auch optionale Attribute, zum Beispiel kann man die Größe des Bildes direkt festgelegen. Hierfür werden die Attribute width & height genutzt, aber man sollte überlegen ob man hierfür nicht lieber CSS nutzt. Mit dem Attribut title kann man zusätlich noch einen Titel festlegen welcher angezeigt wird wenn man mit dem Mauszeiger über das Bild fährt.

Videos

Videos lassen sich ähnlich wie Bilder mit dem tag <video> einfügen. Auch die Quelle muss wieder mit dem Attribut src angegeben werden. Aber auch der Videocodec sollte festgelegt werden, hierzu wird das Attribut type genutzt. Untertützt werden die Formate "video/mp4" und weitgehend "video/ogg" oder "video/webm".

Es gibt auch wieder optionale Attribute, zum Beispiel kann man eine Steuerungsleiste mit dem Attribut controls aktivieren, dieses benötigt keinen Wert. Mit width & height kann man die Größe wieder vordefinieren. Mit loop wird das Video in Dauerschleife abgespielt und muted stellt den Ton auf stumm. Beiden Attributen muss man jeweils keinen Wert zuweisen. Übrigens sollte man das Attribut autoplay nur verwenden wenn es unbedingt nötig ist. Viele Nutzer empfinden es nämlich als nervig wenn ein Video abgespielt wird sobald sie ein Seite betreten.

Musik

Musik oder Audiodateien lassen sich mit dem tag <audio> einbinden. Natürlich muss wieder die Quelle mit src angegeben werden. Aber auch der Audiocodec sollte mit type festgelegt werden. Unterstützt wird "audio/mp3" und weitgehend "audio/wav". "audio/ogg" sollte man eher nicht verwenden da es nicht so weit verbreitet ist. Auch controls sollte angegeben werden, da es sonst schwierig wird die Audio abzuspielen.

Die Attribute width & height gibt es nicht, dafür aber loop und muted welche die selbe Funktion wie bei Videos haben.
