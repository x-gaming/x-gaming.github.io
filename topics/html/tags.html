<tags> für HTML

Hier ist eine Übersicht mit tags welche nach persönlicher Erfahrung immer / häufig gebraucht werden. Wenn du in der Spalte "Wie?" auf das "|-|" klickst bekommst du nochmal ein Anwedungsbeispiel.

Weiter unten findest du noch eine List mit sogenannten Entities.

tags für die Kopfdaten

tag Funktion Wie?
<!DOCTYPE html> Validiert Dokument (macht es gültig, steht immer in der ersten Zeile) |-|

Code
<head> speichert Kopfdaten (werden nicht auf der Seite angezeigt)
<meta> repräsentiert Metadaten, z.B. Zeichensätze oder "keywords" für Suchmaschinen
<title> Titel der Seite (im Browser Tab)
<link> ermöglicht Zugriff auf externe Dateien, z.B. CSS
<script> bindet externes JavaScript ein
<style> CSS-Style für das jeweilige Dokument

tags für Bereiche, sogenannte "Container"

tag Funktion Wie?
<html> Wurzel Element, umschließt alle anderen tags |-|

Code
<div> bezeichnet einen Bereich ohne weitere Eigenschaften
<span> Bereich für ein einzelnes Wort
<header> für die Hauptüberschrift
<nav> für die Navigation einer Seite
<main> für den Hauptinhalt der Seite
<footer> z.B. für Links / Infos, Partnern / Entwicklern oder Quellangaben

tags für die Strukturierung des sichtbaren Inhalts

tag Funktion Wie?
<h1> - <h6> Überschriften in 6 verschiedenen Größen |-|

Code
<p> Sätze, Texte, Absätze
<q> Zitat
<a> Links zu ex- oder internen Seiten
<code> repräsentiert Code Zeilen
<br> erzwingt einen Zeilenumbruch
<ul> ungeordnete Liste (mit Punkten)
<ol> geordnete Liste (mit Zahlen)
<li> Listeneintrag (braucht vorher <ul> oder <ol>)
<dl> Definitionsliste (nutzt <dt> zum definieren und <dd> zum erklären)

tags für Tabellen

tag Funktion Wie?
<table> Tabelle |-|

Code
<caption> Tabellenbeschreibung
<thead> Tabellenkopf (Bereich)
<tbody> Tabellenkörper (Bereich)
<tfoot> Tabellenfuß (Bereich)
<tr> Tabellenreihe
<th> Kopfzelle der Tabelle (definiert quasi die Spalten z.B. 2× <th> = 2 Spalten)
<td> Zelle einer Tabelle (man braucht pro Reihe so viele wie man Kopfzellen hat)

tags für verschiedene Formulare

tag Funktion Wie?
<form> Bereich für ein Formular |-|

Code
<fieldset> fasst mehrere Formularinhalte zusammen
<legend> Überschrift / Beschreibung von einem <fieldset>
<button> Schalter / Knopf
<label> Beschriftung eines <input> Elements

<select> Auswahlkasten

Entities

Entities zählen nicht wirklich zu den tags aber es ist wichtig einige von ihnen zu kennen. Sie stellen sicher, dass Sonderzeichen richtig dargestellt werden oder bestimmte Formatierungen eingehalten werden. Alle Entities starten mit dem &-Zeichen und enden mit einem Semikolon.

Man sollte generell Sonderzeichen nie per Tastenkürzel einfach in den Html-Code schreiben sondern immer das jeweilige Entity nutzen. Besonders für die Zeichen (>, <, " und &) sollte man ihre Entities verwenden, da der Browser diese Zeichen sonst als Html-Code und nicht als normalen Text ansieht.

Entity Zeichen Funktion
&lt; < Das "Kleiner-als-Zeichen"
&gt; > Das "Größer-als-Zeichen"
&quot; " Das Anführungszeichen
&amp; & Das Und-Zeichen
&nbsp;   Leerzeichen wo ein Zeilenumbruch verboten wird
(nützlich bei Wörtern die auf keinen Fall getrennt werden dürfen)
&copy; © Das Copyright-Zeichen
&reg; ® Das Registriert-Zeichen
&dollar; $ Das Dollar-Zeichen
&euro; € Das Euro-Zeichen
&pound; £ Das Britische Pfund-Zeichen
&sect; § Das Paragraphen-Zeichen
&pi; π Das Symbol für die Kreiszahl Pi
&equals; = Das Gleich-Zeichen
&asymp; ≈ Das Ungefähr-Zeichen
&plus; + Das Plus-Zeichen
&minus; − Das Minus-Zeichen
&times; × Das Mal-Zeichen
&divide; ÷ Das Geteilt-Zeichen
