Grundlagen

Wie schon erwähnt, Html ist eine Auszeichnungssprache d.h. sie gibt den Aufbau/ Inhalt vor. Um dies korrekt zu tun nutzt HTML sogennante "tags" (vom englischen Etikett). Ein tag wird immer zwischen spitzen Klammern (größer/ kleiner-als Zeichen) definiert und jeder tag hat seine eigene Funktion.

Ein <tag> in HTML

Fast jeder tag hat auch einen "schließenden End-tag", dies ist nicht kompliziert denn es wird einfach nur ein Schrägstrich (/) vor den tag geschrieben.
Das ganze sieht dann so aus: <tag> Inhalt </tag> wobei diese nicht zwingend hintereinander stehen müssen, sie können auch in verschiedenen Zeilen stehen. Wichtig zu merken ist, bis auf wenige Ausnahmen besteht jede Auszeichnung aus einem tag-Paar mit dem Inhalt dazwischen.

Es gibt auch sogennante "Standalone-tags", diese benötigen keinen End-tag.

Eltern - Kind Elemente

Man kann / muss manchmal bestimmte tags ineinander verschachteln. Der tag welcher dann der "Ursprung" ist, wird auch als "Eltern-tag" bezeichnet, alle tags welche danach folgen werden "Kind-tag" genannt. Zum Beispiel ist bei einer Tabelle <table> der "Eltern-tag" und die Spalten und Zeilen sind die "Kind-tags". Oder wenn man div nutzt ist dieser der "Eltern-tag" und alle tags welcher in diesem enthalten sind, sind "Kind-tags". Besonders für CSS ist dies wichtig, denn dort wird dies als "Vererbung" bezeichnet und hat eine wichtige Aufgabe.

Attribute und Werte

Jeder tag kann ein Attribut, welches einen festgelegten Wert besitzt haben. Auch mehrere Attribute sind möglich, diese werden dann einfach nur mit einem Leerzeichen voneinader getrennt. Attribute werden immer nur im "Start-Tag" definiert und der Aufbau ist wie folgt: <tag Attribut = "Wert">. Diese Attribute werden genutzt um dem Browser zum Beispiel weitere Informationen zu übergeben oder das Verhalten / die Funktion des jeweiligen tags zu "modifizieren". Es gibt auch tags welche ohne Attribut überhaupt nicht funktionieren.

Einfacher Aufbau

Der einfachste Aufbau enthält gerade mal 4 tags: html, head, title, body plus die Anweisung, dass es sich um ein Html-Dokument handelt.

Also fängt man wie folgt an:

  1. <!DOCTYPE = html> in der ersten Zeile um dem Browser mitzuteilen, dass es sich auch wirklich um ein HTML-Dokument handelt.
  2. in die zweite Zeile direkt das Wurzelelement <html> welches alle anderen tags umschließt, hier sollte man auch direkt schon die Sprache einstellen indem man zum html-tag noch den Ländercode hinzufügt. Für Deutschland sieht dies dann so aus: <html lang ="de">. Außerdem noch den "End-tag", dieser steht immer in der letzten Zeile.
  3. Nun folgt der Kopfbereich der Seite, hierzu wird der tag <head> gebraucht.
  4. Besonders für deutsche Seiten ist der Zeichensatz wichtig, ansonsten werden Umlaute wie ä, ü, ö oder ß vielleicht nicht richtig dargestellt. Dies macht man indem man <meta> verwendet. Diesem gibt man das Attribut "charset" und weißt den Wert "UTF-8" zu. Das ganze sieht dann so aus: <meta charset = "UTF-8">.
  5. Auch noch im Kopfbereich folgt der Titel der Seite, welcher im Tab des Browsers steht. Hierzu nutzt man <title>.
  6. Der nächste tag nach dem Kopf ist <body>, dieser ist für alles sichtbare zuständig und somit folgt hier auch alles weitere.
